# React Weather App

(https://github.com/deepthi513/weather-ap)

<hr />

### Installation Options

```
$ https://github.com/deepthi513/weather-ap
```

```
$ npm install
```

```
$ npm start
```

### Or

```
$ yarn
```

```
$ yarn start
```
